## 浏览器的兼容性
- 网络上已经有成熟的css代码库为我们解决浏览器不一致问题，例如normalize.css、minireset和ress，你可以在你的项目中引用它们。
- 如果你不想使用第三方代码库，你可以使用下面的样式来进行一个非常基本的css reset：
```css
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

上面的代码看起来有些霸道，将所有元素的内外边距都设置为0了，而正是没有了这些默认内外边距的影响，使得我们后面的css设置会更加的容易。
同时box-sizing: border-box也是一个很棒的设置，我们紧接着就会介绍它。
#### 所有元素设置为Border-box
大多数初学者都不知道box-sizing这个属性，但实际上它非常重要。box-sizing属性有两个值：
- content-box（默认） - 当我们设置一个元素的宽度或高度时，就是设置它的内容的大小。所有的padding和边框值都不包含。
> 例如，一个div的宽度设置为100，padding为10，于是这个元素将占用120像素（100+2*10）。
- border-box - padding与边框包含在元素的宽度或高度中，一个设置为width: 100px和box-sizing:border-box的div元素，他的总宽度就是100px，无论它的内边距和边框有多少。

## 垂直对齐
如果你用CSS，则你会有困惑：我该怎么垂直对齐容器中的元素？现在，利用CSS3的Transform，可以很优雅的解决这个困惑：
::: demo [vanilla]
```html
<html>
<div class="div">
    <div>垂直居中</div>
</div>
</html>
<style>
.div{ width: 200px;height: 160px;border: 1px solid; }
.div>div{ width: 80px;height: 50px;border: 1px solid;}
/*垂直居中*/
.div>div{position: relative; top: 50%; -webkit-transform: translateY(-50%);-o-transform: translateY(-50%);transform: translateY(-50%);}
</style>
```
:::

> 使用这个技巧，从单行文本、段落到box，都会垂直对齐。目前浏览器对Transform的支持是需要关注的.
>
> Chrome 4, Opera 10, Safari 3, Firefox 3, and Internet Explorer 9均支持该属性。

## 背景渐变动画
CSS中最具诱惑的一个功能是能添加动画效果，除了渐变，你可以给背景色、透明度、元素大小添加动画。目前，你不能为渐变添加动画，但下面的代码可能有帮助。它通过改变背景位置，让它看起来有动画效果。
::: demo [vanilla]
```html
<html>
    <button class="button">背景渐变动画</button>
</html>
<style>
.button {cursor: pointer;}
.button {
    background-image: linear-gradient(#5187c4, #1c2f45);
    background-size: auto 200%;
    background-position: 0 100%;
    transition: background-position 0.5s;
}    
.button:hover {
    background-position: 0 0;
}
</style>
```
:::

## 用CSS动画实现省略号动画
::: demo [vanilla]
```html
<html>

<style>
.loading:after {
    overflow: hidden;
    display: inline-block;
    vertical-align: bottom;
    animation: ellipsis 2s infinite;
    content: "\2026"; /* ascii code for the ellipsis character */
}
@keyframes ellipsis {    
    from {
        width: 2px;
    }
    to {
        width: 15px;
    }
}
</style>
```
